This is an api of fast food orders </b>
Functions: are :</b>
GET /orders  Get all the orders. </b>
GET /orders/<orderId>  Fetch a specific order </b>
POST /orders  Place a new order. </b>
PUT /orders/<orderId>  Update the status of an order. </b>
